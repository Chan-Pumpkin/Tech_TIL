# FormattingTag(포매팅 태그)
포매팅 라이브러리를 사용하면 쉽게 원하는 형태로 숫자, 날짜, 문자열을 표시할 수 있다.

## taglib
```jsx
<%@ taglib prefix="fmt" uri=" http://java.sun.com/jsp/jstl/fmt ">
```   
    
## Formatting Tag library   
1. <fmt:timeZone>   
지정한 국가의 시간을 지정하는 태그     
태그를 열고 닫는 영역 안에서만 적용된다.   

2. <fmt:setTimeZone>    
지정한 국가의 시간을 지정하는 태그  
    
3. <fmt:formatNumber>    
표시할 숫자의 형식을 지정

4. <fmt:formatDate>    
지정한 형식의 날짜를 표시

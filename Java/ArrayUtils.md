# ArrayUtils

## 의존성 추가
```xml
<dependency>
    <groupId>org.apache.commons</groupId>
    <artifactId>commons-lang3</artifactId>
    <version>3.11</version>
</dependency>
```

## 두 값 사이에 값 삽입
```java
int[] testArray = ArrayUtils.insert(1, array, 20);
```

## 배열의 요소를 섞기
```java
ArrayUtils.shuffle(array);
```

## 1. 문자열 결합

### 문자열 결합이란

- 다음과 같이 문자열 데이터를 결합하는 연산

```sql
'ABC'||'1234' -> 'ABC1234'
```

- 2개의 열 데이터를 모아서 1개의 열로 처리하고 싶은 경우에 자주 사용

### 문자열 결합 연산자 제품 차이

문자열을 결합하는 연산자에는 데이터베이스 제품마다 방언이 있으며, 차이가 있다.

| 연산자/함수 | 연산 | 데이터베이스 |
| --- | --- | --- |
| + | 문자열 결합 | SQL Server |
| || | 문자열 결합 | Oracle, DB2, PostgreSQL |
| CONCAT | 문자열 결합 | MySQL |

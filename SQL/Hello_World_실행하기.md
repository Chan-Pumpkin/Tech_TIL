## SELECT * FROM 테이블명

```sql
SELECT*FROM 테이블명;
```
![220303](https://user-images.githubusercontent.com/62877858/156570234-0fae9ec7-d612-45e0-aac2-f75927c536f2.GIF)

예약어 : SELECT, *, FROM      
테이블명 : sampletable      

## 대소문자 구별

예약어와 데이터베이스 객체명은 대소문자를 구별하지 않는다.

```sql
select * from sampleTable;
SelecT * from sampletable;
SELECT * FROM SAMPLETABLE;
```
